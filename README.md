
STM32智能拐杖功能如下：
1.	环境采集：光敏传感器采集环境光照强度、超声波传感器检测与前方障碍物距离、MPU6050陀螺仪检测是否发生跌倒、GPS模块检测经纬度信息。
2.	显示功能：屏幕显示当前时间、光照强度、与前方障碍物距离和经纬度信息
3.	模式切换：通过按键可以切换手动模式和自动模式
4.	自动模式：自动模式下如果环境光照强度小于阈值则开启LED照明灯，如果超声波检测距离小于阈值则蜂鸣器报警提醒、如果发生跌倒事故则蜂鸣器报警提醒且手机APP收到信息“跌倒请注意！”。
5.	手动模式：手动模式下可通过按键控制LED照明灯和蜂鸣器报警
6.	阈值调节：可以通过按键进入系统设置界面，可设置当前时间、光照阈值、距离阈值
7.	蓝牙APP：通过蓝牙APP可接收当前光照情况、超声波与前方障碍物距离、GPS经纬度，如果发生跌倒事故则接收的信息在最后一行显示“跌倒请注意”，可以通过按钮下发控制指令，按钮次序为：A模块、B灯光、C报警
